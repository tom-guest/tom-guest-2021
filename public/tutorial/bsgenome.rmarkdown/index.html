Introduction

In this tutorial we will use the BSgenome package to create an R package that contains the genome of your favourite organism. This sounds scary, but it is very easy to do and makes interacting with the genome much simpler…

The BSgenome package provides a framework for interacting with genome information in R.

Install BSgenome package

if (!requireNamespace("BiocManager", quietly = TRUE)) install.packages("BiocManager") BiocManager::install("BSgenome", version = "3.8")
